What is a State Machine?

A state machine is a way to process input by transitioning between different states. For example:

  • If the input is a letter, the state is WORD.
  • If the input is a digit, the state is NUMBER.
  • If the input is a space, the state resets.

Try It: A Minimal State Machine

Copy and paste this code into your IDE, run it, and see how it works!


public class SimpleStateMachine {
    public static void main(String[] args) {
        String input = "a 1 b";
        System.out.println("Input: " + input);

        String state = "START";

        for (char c : input.toCharArray()) {
            if (Character.isLetter(c)) {
                state = "WORD";
                System.out.println("State: " + state + ", Char: " + c);
            } else if (Character.isDigit(c)) {
                state = "NUMBER";
                System.out.println("State: " + state + ", Char: " + c);
            } else if (Character.isWhitespace(c)) {
                state = "START";
                System.out.println("State: " + state + ", Char: [space]");
            }
        }
    }
}

Steps:

  1. Open your IDE (e.g., IntelliJ, Eclipse, or VS Code).
  2. Create a new Java class called SimpleStateMachine.
  3. Copy and paste the code above into the class.
  4. Run the program.

Expected Output:


Input: a 1 b
State: WORD, Char: a
State: START, Char: [space]
State: NUMBER, Char: 1
State: START, Char: [space]
State: WORD, Char: b

What Just Happened?

The program processes each character in the input and changes its state based on the character:

  • When it sees a letter, it enters the WORD state.
  • When it sees a digit, it enters the NUMBER state.
  • When it sees a space, it resets to the START state.
