What are Token Types?

Token types are categories for the tokens that a lexer produces. For example:

  • WORD: A sequence of letters and numbers (e.g., hello, world123).
  • NUMBER: A sequence of digits (e.g., 42, 3.14).
  • ENDOFLINE: A marker for the end of a line.

Try It: Defining Token Types

Copy and paste this code into your IDE, run it, and see how it works!


public class TokenTypes {
    public enum TokenType { WORD, NUMBER, ENDOFLINE }

    public static void main(String[] args) {
        TokenType type1 = TokenType.WORD;
        TokenType type2 = TokenType.NUMBER;

        System.out.println("Type 1: " + type1);
        System.out.println("Type 2: " + type2);
    }
}

Steps:

  1. Open your IDE (e.g., IntelliJ, Eclipse, or VS Code).
  2. Create a new Java class called TokenTypes.
  3. Copy and paste the code above into the class.
  4. Run the program.

Expected Output:


Type 1: WORD
Type 2: NUMBER

What Just Happened?

The program defines an enum called TokenType with three possible values: WORD, NUMBER, and ENDOFLINE. It then creates two variables of type TokenType and prints their values.
