Try It: A Simple Lexer

Copy and paste this code into your IDE, run it, and see what happens!


public class SimpleLexer {
    public static void main(String[] args) {
        String input = "3 + 5 * 2";
        System.out.println("Input: " + input);

        // Split the input into tokens
        String[] tokens = input.split(" ");

        // Print each token
        for (String token : tokens) {
            System.out.println("Token: " + token);
        }
    }
}

Steps:

  1. Open your IDE (e.g., IntelliJ, Eclipse, or VS Code).
  2. Create a new Java class called SimpleLexer.
  3. Copy and paste the code above into the class.
  4. Run the program.

Expected Output:


Input: 3 + 5 * 2
Token: 3
Token: +
Token: 5
Token: *
Token: 2

What Just Happened?

The program split the input "3 + 5 * 2" into tokens using spaces as separators. Each token is printed to the console.
